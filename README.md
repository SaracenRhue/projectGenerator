# projectGenerator

This tool generates boilerplate code for multiple project types.

## Setup

```bash
bash -c "$(https://raw.githubusercontent.com/SaracenRhue/projectGenerator/main/setup.sh)"
```

## Features

create boilerplates for:

* C++ projects
* Docker projects
* Python projects
* static websites
* React projects
* NodeJS projects
* elecron projects
